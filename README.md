# 🧪 SauceDemo UI Test Automation

Automated UI tests for [saucedemo.com](https://www.saucedemo.com) using Java, Selenium, TestNG, and the Page Object Model (POM) design pattern.

---

## 🚀 Tech Stack

- Java 21
- Selenium 4
- TestNG
- WebDriverManager
- Maven
- Page Object Model (POM)
